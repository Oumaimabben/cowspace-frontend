CowSpace Frontend
📌 Project Overview

CowSpace is a web application for managing reservations and users in a coworking space environment.
This project is built using Angular and provides an intuitive interface for users and administrators to manage bookings, rooms, and accounts.

The application allows users to:

Create an account

Login securely

Browse available rooms

Make reservations

Administrators can:

Manage rooms

Manage users

Monitor reservations

🚀 Technologies Used

Frontend Framework: Angular

Language: TypeScript

Styling: SCSS

Architecture: Component-Based Architecture

Version Control: Git

⚙️ Installation

Clone the repository:

git clone https://github.com/Oumaimabben/cowspace-frontend.git

Go to the project folder:

cd cowspace-frontend
cd client
Install dependencies:

npm install

Run the project:

ng serve

Open in browser:

http://localhost:4200
